How To Use

There are two facets of this hack. One for the daily puzzle, one for the average puzzles. The normal puzzle solver requests the letters available to you, seperated by commas (i.e. o, r, w, d), and will return a list of all words possible. The daily puzzle on the other hand, will request the letters available to you, as with the normal, but will also request the word you want to find. For example, if the word is four letters and ending in "d", type in "---d".

Note: The dictionary used in this hack may include some words not included in the dictionary used by Wordscapes, and vice versa, nevertheless, these should be few and far between.
